# 人格评估网页

一个可部署到 Vercel 的中文单页人格评估网页。测评结果在浏览器本地计算，不需要登录、后端 API 或数据库。

## 功能

- 20 道五分制题目
- 五个原创人格维度：开放探索、稳定执行、社交能量、协作同理、情绪韧性
- 分步沉浸式答题，一次显示 1 道题
- 支持数字键 `1-5` 快速选择
- 支持上一题、下一题、回看修改和重新测试
- 自动在本地浏览器保存答题进度，可手动清除本次答案
- 实时洞察面板展示五维初步表现
- 三套可切换主题：晨雾浅色、夜航深色、纸本暖色
- 支持反向题计分
- 结果归一化为 0-100 分
- 展示雷达图、总体摘要、最高维度、可发展维度和每个维度建议
- 包含自我探索用途免责声明

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物会输出到 `dist`。

## 测试

```bash
npm test
```

当前测试覆盖评分归一化、反向题、预览结果、分数分层和完成度计算。

## 代码检查

```bash
npm run lint
npm run format
```

`lint` 会检查源码尾随空白和文件末尾换行；`format` 会做对应的轻量修正，不依赖额外格式化包。

## 报告导出

结果页支持在浏览器本地生成文本报告，并可展开查看每题答案、所属维度和反向题计分处理。报告生成过程不上传数据。

## 项目边界

本项目是自我探索工具，不是正式心理测量量表；没有常模、信效度验证或临床解释能力。结果不应用于医学/心理诊断、招聘筛选、职业决策或任何高风险场景。

## 部署到 Vercel

在 Vercel 导入该仓库后使用默认 Vite 配置：

- Build Command: `npm run build`
- Output Directory: `dist`

项目不需要配置环境变量。
